Class & Object : Class is a collection of data member and member function

Object : Object is instance of class through the object we can access all the properties of class

    (2) Private
    (3) Protected


Constructor : Constructor is Special Method that can call automatically when Object is created.(Same name as class name).Constructor is always display first.
